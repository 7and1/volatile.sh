2024

December 2024
Initial Public Launch NEW

volatile.sh is now publicly available! Here's what we're launching with:

  • Zero-knowledge secret sharing with AES-256-GCM encryption
  • Burn-after-reading functionality
  • Flexible TTL options (5 minutes to 7 days)
  • Rate limiting to prevent abuse
  • RESTful API for programmatic access
  • Fully open-source (MIT licensed)
  • Self-hosting support via Cloudflare Workers

Upcoming

Planned Features

We're actively working on these features:

  • File upload support (encrypted file sharing)
  • Optional password protection for additional security
  • CLI tool for terminal-based secret sharing
  • Browser extensions for quick access
  • Webhook notifications when secrets are read
